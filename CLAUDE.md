# Jijñāsā Adhikaraṇa — Project Instructions

## Project Context
This is a Sanskrit thesis project analyzing the Dvaita critique of Advaita in Brahma Sūtra 1.1.1.

## Critical Rules
- **Tone**: Respectful, dignified scholarly discourse. Never use harsh or abusive language.
- **Position**: The thesis demonstrates the hollowness of Dvaita arguments against Advaita, establishing Advaita's logical superiority. Approach: "agree to disagree."
- **Language**: Classical Sanskrit is the primary output language. When AI capability falls short, the human scholars will compensate.
- **Font**: VijayaDV PUA characters must never be stripped or normalized.
- **Transliteration**: Use IAST for romanized Sanskrit (ā, ī, ū, ṛ, ṣ, ś, ṇ, ṃ, ḥ etc.)

## Source Texts
1. **Nyāya Sudhā** (Jayatīrtha) + prominent commentaries — Dvaita primary source
2. **Śārīraka Mīmāṃsā Bhāṣya** (Śaṅkarācārya) + **Ānandagiri Ṭīkā** — Advaita primary source
3. **Dhvānta Nirāsa** — Advaita counter-refutation
4. Other useful treatises as identified
